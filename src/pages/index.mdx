# How Oembed works in this Gatsby Remark Plugin
Oembed transforms oembed links into its corresponding embed code. The sparkliest way for a developer to Embed his flashiest finds from for example CodePen, CodeSandbox, Twitter, Instagram, YouTube, Vimeo and SoundCloud.

This is an early version of the plugin. Let me know if you have problems or questions by submitting an issue.
